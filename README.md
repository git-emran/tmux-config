# Welcome to my Tmux Config.

This is my custom setup for my tmux.

## Table of Contents

My tmux config is seperated into different modules for maintainability. The following files are:

#### theme.conf

All the colorschemes are here. I mainly use "Solarized Osaka.

#### macos.conf

Native macOS Terminal overrides.

#### statusline.conf

Statusline design that matches my nvim statusline.

#### utility.conf

All the keybinds for window splits and split navigation.
